# schemas / agent_manifest

Status: active machine-contract family by AAI-CP5  
Updated: 2026-05-18T00:00:00+02:00

Files in this folder are grouped for findability only. Semantic authority and default status are governed by active OFARM law and the contract indexes at `03_machine_contracts/`.

AAI-CP5 promotes bounded agentic capability/tool-manifest and capability-honesty contracts for declared tools, approval requirements, external-call posture, trace retention, redaction, data-learning posture, semantic preconditions, declared hints, and readiness-claim limits.

## Non-claims

These schemas do not grant authority, prove runtime capability, allow hidden external calls, or establish production AI-agent readiness. Manifest declarations remain auditable statements constrained by active authority and conformance evidence.

Examples for this family live under `04_implementation_and_conformance/examples_and_fixtures/examples/machine_contracts/agent_manifest/` and are mapped by `03_machine_contracts/EXAMPLE_SCHEMA_MAP.json`.
