# Query and Capability Performance Envelope Draft Schemas

Status: draft/non-default machine-contract family.

These schemas support `02_accepted_rfcs/OFARM_Performance_and_Explainable_Current_State_Evidence_RFC_v0_1.md` as draft extensions to QueryPlanIR and Capability Manifest self-description.

They do not modify current/default `QueryPlanIR` or `Capability Manifest` schemas, do not promote any draft machine contract to current/default, and do not change active baseline law.

The family covers:

- `QueryPlanIRPerformanceEnvelopeExtension`
- `CapabilityManifestExplainableCurrentStatePerformanceEnvelopeExtension`

Benchmark artifacts remain runtime evidence only. These draft extensions do not let cost-aware planning downgrade freshness, traceability, review, refusal, or recomputation requirements.
