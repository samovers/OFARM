# Explainable Current-State Evidence Draft Schemas

Status: draft/non-default machine-contract family.

These schemas support `02_accepted_rfcs/OFARM_Performance_and_Explainable_Current_State_Evidence_RFC_v0_1.md`.
They are not current/default machine contracts and must not be used as current/default schemas unless a later currentness-promotion decision explicitly promotes them.

The family covers:

- `MaterializationKey`
- `MaterializationDependencyIndex`
- `MaterializationFreshnessVector`
- `InvalidationEvaluationTrace`
- `TraceExpansionPolicy`
- `TraceExpansionRequest`
- `TraceExpansionResult`
- `MaterializationCostProfile`
- `RuntimeQueryMixProfile`
- `RuntimeMaterializationBenchmarkRun`
- `RuntimeStorageAmplificationReport`
- `ExplainableCurrentStateCapabilityClaim`

These draft schemas do not modify active baseline law, do not promote machine contracts to current/default, do not make projections authoritative, and do not make benchmark artifacts canonical farm truth.
