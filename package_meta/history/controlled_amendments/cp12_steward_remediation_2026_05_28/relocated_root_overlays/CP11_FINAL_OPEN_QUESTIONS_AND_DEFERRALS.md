# CP11 Remaining Open Questions and Explicit Deferrals

## Remaining open questions

1. Which sustainability metric profiles should become first-class examples after CP11 acceptance?
2. Which external sustainability/certification schemes should be reviewed first as semantic profiles or attestation wrappers?
3. How much farmer-facing disclosure is usable without creating burden/noise?
4. What runtime policy engine should execute charter gates?
5. Which CP11 fixtures should become permanent conformance runners rather than starter fixtures?
6. What is the first pilot scenario for validating CP11 claim-basis and output-qualification behaviour?

## Explicit deferrals

### CP12 — Cyber-Physical Mission Envelope

Deferred concepts include RobotMission, MissionPlan, GeoFence, NoGoZone, CommandEnvelope, CommandSignature, AutonomyLevel, EmergencyStopPolicy, HumanOverridePolicy, MissionVerification, NearMissEvent, and PhysicalSafetyIncident.

### CP13 — Learning, Experimentation, and Farm Memory

Deferred concepts include Hypothesis, TrialDesign, ControlCondition, TreatmentPlan, CausalEstimate, FarmMemoryEntry, SeasonalLearningSummary, LearningPromotionDecision, and ExperimentRollbackTrigger.

### CP14 — Farm-to-Farm Intelligence Boundary

Deferred concepts include FarmIntelligenceShareGrant, RegionalAlert, BenchmarkDelta, FederatedLearningContribution, ModelUpdateArtifact, AggregationFloor, DerivativeUsePolicy, ReidentificationRiskAssessment, and CommercialUseRestriction.

### CP15 — Agentic Software Delivery Governance

Deferred concepts include GeneratedSoftwareArtifact, AdapterGenerationRequest, SemanticMappingCandidate, ConformanceTestPlan, BuildProvenance, SBOMReference, DeploymentCandidate, CanaryResult, RollbackPlan, and GeneratedCodeTrace.

## Scope exclusion

CP11 does not expand OFARM crop-farming operational law into livestock identity, welfare, feeding, treatment, herd/flock, movement, or animal-health law.
