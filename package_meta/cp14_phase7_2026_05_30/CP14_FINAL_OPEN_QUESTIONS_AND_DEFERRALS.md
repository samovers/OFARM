
# CP14 Final Open Questions and Deferrals

Status: final CP14 open-questions note

## Explicit deferrals

CP14 defers:

- CP15 generated-software and model-deployment governance;
- OFARM Social constitution;
- OFARM Exchange constitution;
- public benchmark product law;
- production federated-learning platform implementation;
- generic reputation/social graph law;
- livestock-specific cross-farm intelligence;
- legal/privacy/certification/insurance/advisory readiness;
- current/default schema promotion.

## Remaining open questions

- Which privacy/re-identification risk method should be used for first external review?
- Which aggregation floors should be profile-specific rather than globally declared?
- How should already-trained models respond to revocation? Full detail belongs to CP15.
- How should public-authority regional alerts be incorporated without becoming hidden OFARM law?
- What boundaries should OFARM Social and OFARM Exchange use if later created as sister platforms?
- Which jurisdiction-specific cross-border data-space constraints should be profiled first?
