# FURS FFS Sprayer-Inspection Downloads

Source: https://spletni2.furs.gov.si/FFS/FFSNaprave/

Observed download formats on the index page:

- xls: legacy Microsoft Excel file; 2026 sample identified by file(1) as CDFV2 Microsoft Excel.
- txt: semicolon-delimited text with quoted header fields; 2026 sample saved in furs_samples.
- xml: XML data file; 2026 sample saved in furs_samples.
- xsd: XML schema file; 2026 sample saved in furs_samples.
- OPIS_POLJ.pdf: field description PDF; extracted text saved as extracts/furs_ffsnprave_opis_polj.txt.

The index page lists these four FFSPreglediNaprav formats for 2026 back through 2001 plus a Do2000 period.
