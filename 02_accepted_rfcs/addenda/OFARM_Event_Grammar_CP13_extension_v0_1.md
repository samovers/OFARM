# OFARM Event Grammar CP13 extension v0.1

Status: accepted/merged CP13 addendum; subordinate to active baseline and CP13 RFC.

Candidate learning event families:

- LearningHypothesisSubmitted
- ExperimentProtocolProposed
- ExperimentProtocolApproved
- TrialDesignRegistered
- OutcomeObservationRecorded
- LearningEvidenceBundleAssembled
- LearningEvaluationRecorded
- CausalEstimateRecorded
- LearningPromotionDecisionRecorded
- FarmMemoryEntryActivated
- FarmMemoryEntryInvalidated
- FarmMemoryRetrieved
- SeasonalLearningSummaryIssued
- ExperimentRollbackTriggered
- ExperimentExceptionRequested
- ExperimentExceptionApproved

Event presence alone does not create truth, current state, farm memory, compliance fact, mission authority, or deployment authority.
