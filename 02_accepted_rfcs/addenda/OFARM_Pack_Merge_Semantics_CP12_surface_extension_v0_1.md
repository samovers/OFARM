# OFARM Pack Merge Semantics — CP12 mission-surface extension v0.1

Status: accepted/merged CP12 addendum; active below baseline authority. CP12 machine contracts remain draft/non-default.

## CP12 mission pack surfaces

| Surface | Default merge posture |
|---|---|
| MISSION_SAFETY_CONSTRAINT | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_GEOFENCE_POLICY | CONSTRAINT_INTERSECTION or HARD_FAIL |
| MISSION_NO_GO_ZONE_POLICY | CONSTRAINT_INTERSECTION or HARD_FAIL |
| MISSION_ROUTE_POLICY | CONSTRAINT_INTERSECTION or HARD_FAIL |
| MISSION_AUTONOMY_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_EMERGENCY_STOP_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_HUMAN_OVERRIDE_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_LOCAL_FALLBACK_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_COMMAND_INTEGRITY_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_CAPABILITY_COMPATIBILITY_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_VERIFICATION_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |
| MISSION_OUTPUT_QUALIFICATION_POLICY | STRONGEST_REQUIREMENT or HARD_FAIL |

Packs may make mission policy stricter. They may not create mission dispatch authority, weaken emergency stop, weaken command integrity, or make vendor payload validity equivalent to OFARM mission authority.
