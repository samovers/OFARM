# OFARM Authority Action Matrix CP13 extension v0.1

Status: accepted/merged CP13 addendum; subordinate to active baseline and CP13 RFC.

Candidate CP13 action classes:

- LEARNING_SCOPE_DEFINE
- LEARNING_HYPOTHESIS_SUBMIT
- EXPERIMENT_PROTOCOL_APPROVE
- TRIAL_DESIGN_APPROVE
- OUTCOME_MEASURE_REGISTER
- LEARNING_EVALUATION_REVIEW
- CAUSAL_ESTIMATE_REVIEW
- LEARNING_PROMOTION_APPROVE
- FARM_MEMORY_ACTIVATE
- FARM_MEMORY_INVALIDATE
- FARM_MEMORY_RETRIEVE_HIGH_CONSEQUENCE
- SEASONAL_LEARNING_SUMMARY_APPROVE

Default posture: agents may prepare, evaluate, request evidence, and package review material under authority envelope; agents may not by default approve experiments, activate farm memory, promote causal claims, authorise operations/missions, approve cross-farm use, or deploy models/software.
