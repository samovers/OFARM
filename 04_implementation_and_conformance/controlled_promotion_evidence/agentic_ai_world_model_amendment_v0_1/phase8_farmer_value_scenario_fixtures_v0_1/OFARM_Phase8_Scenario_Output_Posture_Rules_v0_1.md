# OFARM Phase 8 — Scenario Output Posture Rules v0.1

## Output dispositions used in Phase 8

- `ADVISORY_SUMMARY`
- `DRAFT_EVIDENCE_CANDIDATE`
- `DRAFT_DOCUMENT_ASSEMBLY`
- `PREFLIGHT_RESULT`
- `BRIDGECANDIDATE_PROPOSAL_ONLY`
- `EVIDENCENEED`
- `OBSERVATIONREQUEST`
- `REQUEST_DISPLAY_ENVELOPE`
- `PERMISSION_LIMITED_RESULT`
- `BLOCKED_ACTION_TRACE`
- `DISPUTE_RECONSTRUCTION_PACKET`
- `BENCHMARK_ADVISORY_ONLY`

## Rule

A Phase 8 fixture must not treat any of these output dispositions as accepted Compliance Twin truth unless it explicitly routes through normal OFARM evidence, authority, review, promotion, output approval, and filing gates.
