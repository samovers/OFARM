# OFARM Phase 8 — Attached Research Analysis v0.1

## Research signals used

The attached research identifies the highest-value crop-farm agentic scenarios as governed assistance patterns rather than generic autonomous farming. It emphasizes:

- evidence capture and compliance assembly;
- contractor and offline work reconciliation;
- advisory scenario planning;
- storage and lot-lineage reconstruction;
- buyer readiness and selective disclosure;
- dispute reconstruction;
- controlled benchmarking and learning;
- data sovereignty, redaction, revocation, and explicit training-use policy;
- event-first traceability for lot and buyer-facing workflows;
- local-knowledge-aware evidence and observation requests.

## Phase 8 application

Phase 8 converts those signals into end-to-end fixtures. Each fixture is designed to expose whether agentic OFARM can help a farmer while preserving the following boundaries:

1. agent identity is not authority;
2. agent run success is not governance success;
3. advisory/world-model output is not compliance truth;
4. missing-information requests are not evidence;
5. handoff transfers context but not authority;
6. buyer sharing is scoped and revocable;
7. benchmarking and learning require explicit policy and aggregation safeguards.

## Held decisions

Phase 8 does not define UI, mobile workflow, implementation API, telemetry backend, identity stack, policy engine, or model provider. Those remain implementer choices constrained by OFARM law and future conformance gates.
