# OFARM Phase 8 — Farmer-Value Conformance Fixture Plan v0.1

## Purpose

This plan defines pre-implementation scenario fixtures that an eventual implementer should turn into executable tests.

## Scenario families

The fixture set covers daily attention, evidence capture, compliance prevention, contractor/offline work, local memory, lot and storage traceability, selective disclosure, dispute reconstruction, advisory world-model reasoning, agent handoff, and controlled benchmarking.

## Future execution expectations

A future implementation should execute each scenario through:

1. public operation descriptors;
2. preflight/dry-run/explain surfaces;
3. authority and sharing enforcement;
4. result qualification envelopes;
5. trace retrieval;
6. agent run traces;
7. blocked-action traces;
8. output disposition checks.
