# OFARM Phase 8 — Negative Case Register v0.1

Each scenario has a hostile companion case. The required blockers are:

1. Daily brief hides stale or permission-limited current state.
2. Evidence capture agent treats label photo as accepted evidence without review.
3. Compliance steward converts failed preflight into silent waiver.
4. Contractor offline sync overwrites accepted history after revocation.
5. Local memory note becomes compliance fact.
6. Buyer-readiness agent infers lot status from buyer system alone.
7. Sharing agent over-discloses beyond SharingGrant.
8. Dispute reconstruction suppresses competing evidence interpretation.
9. World-model scenario becomes current-state materialization.
10. Handoff transfers authority from scouting agent to planning agent.
11. Benchmarking/training uses farm data without explicit learning policy and aggregation safeguards.
