# OFARM Agentic Crop-Farm Scenario Catalogue v0.1

Status: DRAFT_COMPANION_CANDIDATE_NOT_PROMOTED

## Scenario catalogue

The recommended crop-farm agentic scenario catalogue includes:

- daily farm command brief;
- evidence capture and compliance assembly;
- compliance steward preflight;
- contractor and offline sync;
- local memory and farmer knowledge;
- lot/storage buyer readiness;
- sharing and data sovereignty;
- dispute reconstruction;
- advisory world-model planning;
- agent-to-agent handoff;
- controlled benchmarking and learning.

## Boundary

This catalogue is a scenario catalogue, not a product roadmap or active law. It should be used to test whether implementation choices preserve OFARM authority, evidence, freshness, sharing, pack, output, and twin boundaries.
