# OFARM Phase 8 — End-to-End Scenario Rules v0.1

## Required fixture sections

A Phase 8 fixture must include:

- scenario identity;
- farmer value goal;
- relevant OFARM areas;
- phase dependencies;
- actors and agent roles;
- setup assumptions;
- workflow steps;
- generated objects;
- expected output dispositions;
- required guardrails;
- must-not-happen cases;
- expected conformance assertions;
- runtime and promotion status.

## Positive path

The positive path should show how agents reduce farmer burden, clarify risk, collect evidence, prepare review, or improve sharing without becoming hidden governance.

## Negative path

The negative path should prove that the same workflow cannot be abused to create hidden truth, silent authority transfer, stale high-consequence output, or excessive farmer task burden.
