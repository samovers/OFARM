## Purpose

Phase 8 creates a farmer-value scenario fixture pack that connects Phases 3–7 into end-to-end crop-farm situations. It is a pre-implementation conformance and design-quality package, not active OFARM law.

The phase tests whether the agentic concepts are useful in farmer-facing workflows while preserving OFARM discipline:

- agent identity and sponsor accountability;
- run envelope, trace, and handoff boundaries;
- tool/capability manifest limits;
- Advisory-only world-model state;
- bounded EvidenceNeed and ObservationRequest objects;
- authority, evidence, freshness, sharing, pack, output, and twin-separation gates.
