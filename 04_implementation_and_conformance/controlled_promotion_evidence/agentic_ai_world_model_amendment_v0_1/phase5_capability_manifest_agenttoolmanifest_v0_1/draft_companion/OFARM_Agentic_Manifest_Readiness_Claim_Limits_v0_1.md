# OFARM Agentic Manifest Readiness Claim Limits v0.1 Candidate

Status: SUPPORTING_DRAFT_NOT_ACCEPTED

## Rule

No pre-implementation OFARM package may claim runtime agentic readiness merely because schemas, examples, or manifests exist.

## Allowed pre-implementation statements

- Declared candidate support
- Static schema validation
- Static example validation
- Architect-review readiness

## Disallowed pre-implementation statements

- Runtime conformance passed
- Production-ready
- Two-agent compatible
- Certified for autonomous compliance decisioning
- World-model compliance-ready
- External-standard certified
