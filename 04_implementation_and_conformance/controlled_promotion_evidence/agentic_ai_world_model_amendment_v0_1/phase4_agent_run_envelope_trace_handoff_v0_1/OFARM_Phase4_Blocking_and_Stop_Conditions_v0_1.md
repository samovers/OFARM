# OFARM Phase 4 Blocking and Stop Conditions v0.1

Status: supporting review note; not promoted

Runs block, degrade, request review, request approval, or stop when actorship is invalid, envelope is missing/expired, scope/twin/action/tool/write surface exceeds the envelope, preflight fails, approval is missing, revocation occurs, current state is stale, sharing is exceeded, evidence is insufficient, conflicts would be hidden, or output disposition is not allowed.
