# OFARM Phase 4 Query, Current-State, and Freshness Basis Requirements v0.1

Status: supporting review note; not promoted

Agent runs relying on query/current state must preserve QuerySpecification refs, QueryPlanIR refs, MaterializationResult refs, ContextSnapshot refs, PackActivationSet refs, result qualifications, freshness state, and permission/redaction/advisory markers. Daily briefs and compliance outputs must not strip these qualifications.
