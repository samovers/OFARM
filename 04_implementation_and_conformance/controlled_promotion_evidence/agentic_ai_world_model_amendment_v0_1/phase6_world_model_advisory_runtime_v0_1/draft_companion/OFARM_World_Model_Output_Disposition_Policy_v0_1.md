# OFARM World Model Output Disposition Policy v0.1

## Status

Draft companion candidate. Supporting only.

## Disposition policy

Every world-model output must have a disposition. A disposition is not a promotion decision; it is a safety classification that tells the platform and user what the output can and cannot be used for.

## High-consequence outputs

Any attempt to use world-model output for compliance promotion, current-state materialization, attestation, filing, output freezing, or external disclosure must pass separate OFARM authority, evidence, freshness, sharing, review, and publication gates.
