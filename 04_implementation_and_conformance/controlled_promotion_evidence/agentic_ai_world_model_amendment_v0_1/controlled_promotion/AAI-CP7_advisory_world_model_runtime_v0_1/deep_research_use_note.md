# CP7 Deep Research use note

The attached Deep Research report was used to bound CP7. Its key applied findings were:

- simulation and digital-twin state must remain conditional, uncertainty-bearing, and lifecycle-governed;
- `WorldModelRun` needs model identity, calibration reference, assumptions, scenario scope, validity windows, invalidation, uncertainty, and reconciliation routes;
- `WorldModelState` is especially sensitive because users treat state-like objects as truth-like objects;
- CP7 should promote contracts first but keep stronger readiness claims out of baseline law.

Result: CP7 promotes a contract layer, not a readiness claim.
