# AAI-CP4 — Agent run trace and handoff controlled promotion v0.1

Status: controlled promotion completed; active baseline/RFC/machine-contract subset promoted.

CP4 promotes the bounded agent-run layer needed after CP2 public surfaces and CP3 sponsor-bound actorship. It makes software-agent runs bounded by `AgentRunEnvelope`, observable through `AgentRunTrace`, and auditable through tool-invocation, output-disposition, blocked-action, approval, freshness, input-bundle, stop-condition, and handoff records.

CP4 does not claim runtime AI-agent readiness or two-agent compatibility. It promotes active contract shapes and conformance expectations only.
