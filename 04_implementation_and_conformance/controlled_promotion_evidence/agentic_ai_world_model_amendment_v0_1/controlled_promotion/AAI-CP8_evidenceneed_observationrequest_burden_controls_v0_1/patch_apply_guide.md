# AAI-CP8 patch apply guide

Apply this patch over `OFARM2_2026-05-16_agentic_ai_controlled_promotion_cp7_v0_1`.

AAI-CP8 promotes a bounded request-layer contract set for EvidenceNeed, ObservationRequest, and burden/noise/display controls. It does not promote requests as evidence, obligations, or blockers by themselves, and it does not claim farmer-facing workflow readiness, production readiness, autonomous compliance decisioning, legal advice, or external-standard readiness.
