# OFARM Phase 9 Negative Case Register v0.1

Phase 9 registers hostile cases where a plausible agentic farming platform might appear convenient but would weaken OFARM.

The suite rejects:

- agent identity without attestation
- stale sponsor authority replay
- handoff-carried authority
- tool hints treated as trust
- external data auto-promotion
- hidden cache/vector/projection truth
- pack semantic mutation
- query widening and sharing bypass
- stale or invalid world-model use
- memory-to-truth leakage
- benchmark deanonymisation
- missing blocked-action traces
- offline late-sync overwrite
- BridgeCandidate auto-promotion
- output preview publication
- EvidenceNeed-as-evidence
- ObservationRequest-as-fact
- tool success as governance success
- manifest readiness overclaim
- agent self-review
- agent pack activation
- stale current state in high-consequence output
