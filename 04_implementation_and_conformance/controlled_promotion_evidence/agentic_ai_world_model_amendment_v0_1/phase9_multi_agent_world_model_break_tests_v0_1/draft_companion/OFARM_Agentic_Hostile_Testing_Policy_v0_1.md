# OFARM Agentic Hostile Testing Policy v0.1 — Draft Companion Candidate

Agentic OFARM readiness requires hostile testing. Happy-path examples are insufficient.

A hostile test is valid when it attempts a plausible shortcut and expects OFARM to block, downgrade, recheck, redact, or retain as candidate material rather than accept the shortcut.

Every hostile test should assert both:

- the forbidden outcome did not happen
- the blocked attempt is visible in trace evidence
