# OFARM Minimum Capture Profile Draft v0.1

Date: 2026-05-13  
Status: `DRAFT_NON_DEFAULT`

This folder contains a non-default pilot fixture for ordinary-farm minimum capture. It is deliberately not baseline law and not a production default.

Use it to test app flows that preserve missing-data, late-bound, and review-required states without blocking all ordinary farm capture.

Do not use it to bypass evidence, authority, product identity, unit, area, actor, or high-consequence output gates.
