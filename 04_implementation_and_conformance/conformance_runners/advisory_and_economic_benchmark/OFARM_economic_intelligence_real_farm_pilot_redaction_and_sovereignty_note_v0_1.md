# OFARM economic intelligence real-farm pilot redaction and sovereignty note v0.1

## Purpose

Run one farm pilot without violating the agreed OFARM sovereignty posture.

## Required redaction posture

Keep these as stable but redacted refs:
- farm id
- field id
- facility id
- operation id
- contractor extract id
- settlement extract id

Remove or avoid exporting:
- personal employee names unless operationally necessary
- government ids / tax ids
- bank account numbers
- raw invoice PDFs unless needed as separate evidence objects
- payroll line items not needed for the lane
- buyer identities if the pilot can use coded refs instead

## Sharing rule

The pilot package is farm-private by default.
Sharing the pilot result does **not** imply authority to edit farm truth.

## Storage rule

The pilot package is an Advisory execution artifact.
It must not become the authoritative store for economic truth.
