# OFARM Practical Farm Contracts v0.1

Date: 2026-05-13  
Status: active supporting implementation/conformance candidate

## Purpose

This folder contains Phase 6 practical farm contract examples for AI-agent-ready platform development.

It covers:

- governed calculation/unit previews
- formula catalogue and unit-conversion trace examples
- identity resolution and duplicate import posture
- source-fidelity envelopes
- FMIS shadow import candidates and loss maps
- offline capture and delayed sync replay

## Authority posture

These examples do not create active baseline law. They are fixtures for implementation and conformance. Use them to prevent AI coding agents from hiding formulas, guessing identities, over-promoting imports, or treating offline drafts as accepted truth.
