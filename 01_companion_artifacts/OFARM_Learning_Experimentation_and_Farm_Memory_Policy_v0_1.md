# OFARM Learning, Experimentation, and Farm Memory Policy v0.1

Status: active CP13 companion policy; subordinate to active baseline and accepted RFCs.

This note interprets CP13. It does not override the active baseline or accepted RFCs.

Core policy:

- Learning output is not truth.
- Experiment result is not automatic causal fact.
- Farm memory is not hidden current state.
- Agent memory is not farm memory.
- Causal estimates require method, design, evidence, uncertainty, and review posture.
- Farm-memory retrieval must carry qualification and prohibited-use boundaries.
- Cross-farm learning belongs to CP14.
- Model/software deployment belongs to CP15.

Examples and conformance fixtures should focus on outcome-hacking prevention, missingness, causal uncertainty, farm-memory invalidation, retrieval qualification, CP11/CP12 evidence use, and blocked stronger uses.
